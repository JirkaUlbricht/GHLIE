# Github Label Import/Export

Currently work in progress CLI tool to import and export Gihub labels

## Installation

Install GHLIE via NPM

```js
npm install ghlie
```

## Usage

Use ghlie -h for help

ghlie import|export \<owner\> \<repo\>
